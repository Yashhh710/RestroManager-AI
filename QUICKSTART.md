# ⚡ Quick Start - FlowLens AI Chatbot

Get your AI chatbot running in **30 seconds**!

## 1️⃣ You're Already Set Up!

Your API key is already configured in `.env.local`. You're good to go!

## 2️⃣ Start the Dev Server

```bash
npm install
npm run dev
```

## 3️⃣ Open Your Chatbot

- Go to **ExecutiveDashboard** page
- Look for **AI ASSISTANT** panel on the right
- Start typing!

## 4️⃣ Try These Questions

### First Question
```
"What are my top selling dishes and why do you think they're popular?"
```

### Second Question
```
"I have ₹48,760 daily revenue. What can I do to increase this to ₹60,000?"
```

### Third Question
```
"My 'Veg Pasta' has very low sales. Should I remove it or redesign it?"
```

## 🆘 Quick Troubleshooting

### "API key not configured"
```
→ Check .env.local exists
→ Restart dev server (kill and npm run dev again)
```

### "Chatbot not appearing"
```
→ Press F12 to open DevTools
→ Check Console for errors
→ Clear browser cache (Ctrl+Shift+Delete)
```

### "Getting generic responses"
```
→ Provide more context and numbers
→ Ask specific questions about your business
→ Include metrics from dashboard
```
