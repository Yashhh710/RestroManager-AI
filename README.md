# 🚀 Getting Started

Clone the repository

```bash
git clone YOUR_GITHUB_URL
```

Navigate to the project

```bash
cd RestroManager-AI
```

Install dependencies

```bash
npm install
```

Start the development server

```bash
npm run dev
```

Build for production

```bash
npm run build
```
